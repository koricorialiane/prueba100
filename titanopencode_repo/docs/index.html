Red GSM/EDGE tactica con analitica web profesional

Esta pagina sintetiza el reto docente Protocolo Titan como si fuera un panel de ingenieria listo para defensa, revision tecnica y publicacion estatica. Integra movilidad extrema, planificacion espectral, reutilizacion celular y criterio instrumental de certificacion en una unica experiencia navegable.

La web es estatica, se puede abrir directamente desde el archivo HTML y tambien queda preparada para publicarse en GitHub Pages sin backend.

Que domina en cada escenario

Lectura ejecutiva de los resultados mas importantes.

  • Escenario A: domina la variacion temporal del canal y el impacto del Doppler sobre la estabilidad intra-rafaga.
  • Escenario B: dominan la disciplina espectral, la proteccion co-canal y la correcta interpretacion de las medidas de laboratorio.
  • Este paquete contiene solo la web, las figuras necesarias y el codigo fuente de calculo.

Escenario A · Convoy de alta velocidad

Analisis de Doppler, tiempo de coherencia y comportamiento del canal durante el timeslot GSM.

  • A 50 km/h el Doppler queda en 41.7 Hz y el tiempo de coherencia en 10.15 ms.
  • A 250 km/h el canal sigue superando el timeslot GSM con una relacion Tc/Tslot de 3.52.
  • El peor caso presenta LCR de 200 cruces/s y AFD media de 0.503 ms.
  • El ancho de coherencia estimado ronda 111.1 kHz, util para justificar si el canal se comporta casi plano o ya entra en transicion selectiva.

Escenario B · Campamento base

Planificacion de ARFCNs, BCCH/TCH, reutilizacion celular y criterio de certificacion.

  • Cada celda recibe 6 portadoras y la distancia de reutilizacion alcanza 5.2 km.
  • La relacion D/R vale 3.46 y la C/I estimada se situa en 11.64 dB.
  • Pasar de 100 kHz a 1 kHz reduce el ruido integrado mostrado en 20 dB, pero ralentiza el barrido instrumental.

Codigo del repositorio

Paquete limpio con la pagina HTML y el codigo Python necesario para recalcularla.

  • Abre index.html o docs/index.html para ver la web.
  • El codigo fuente esta en src/protocolo_titan/.
  • Las dependencias minimas estan en requirements.txt.
  • Si quieres recalcular la web, ejecuta python -m protocolo_titan.main con PYTHONPATH=src.
  • La carpeta tests/ contiene pruebas para validar los calculos.
